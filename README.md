# Real-time-Chat-Application

A Real-Time Chat Application built using React.js, Node.js, Socket.IO, and MySQL. The application supports instant messaging, real-time communication, AES-based end-to-end style encryption, secure ciphertext storage, and seamless frontend-backend connectivity without page refresh.

## Features

- Real-time messaging using Socket.IO
- End-to-End style encrypted communication
- Secure ciphertext storage in MySQL
- React.js frontend interface
- Node.js + Express.js backend
- Automatic chat scrolling
- Live multi-user communication
- Responsive UI design

## Technologies Used

- React.js
- Node.js
- Express.js
- Socket.IO
- MySQL
- CryptoJS
